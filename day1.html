What do people paste in the console?

The console is like a quick ‚Äúscratchpad‚Äù where you paste short scripts to inspect the page, test functions, fetch data, or debug problems.

Paste Example 1 ‚Äî Inspect the page

document.title

// 2) Get the first H1 text

// 3) Count all links on the page

Paste Example 2 ‚Äî Fix/verify selectors

btn  // prints null OR the element

// If null, try searching by tag/text/class

document.querySelector(".start")

Paste Example 3 ‚Äî Print arrays nicely

  {id:1, name:"Asha", city:"Varanasi"},

Paste Example 4 ‚Äî Quick micro-tests

function clamp(n, min, max){
  return Math.max(min, Math.min(max, n));
}
console.assert(clamp(5,0,10) === 5, "clamp(5) failed");
console.assert(clamp(-2,0,10) === 0, "clamp(-2) failed");
console.assert(clamp(99,0,10) === 10, "clamp(99) failed");

Paste Example 5 ‚Äî Test fetch quickly

  .then(data => console.log("Fetched:", data))
  .catch(err => console.error("Fetch error:", err));
Then verify it in DevTools ‚Üí Network ‚Üí Fetch/XHR.
